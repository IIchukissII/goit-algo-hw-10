|                | Лимонад   | Фруктовий сік | $→max$ |
|----------------|-----------|---------------|---------------------|
| Вода           | 2         | 1             | 100                 |
| Цукор          | 1         |               | 50                  |
| Лимонний сік   | 1         |               | 30                  |
| Фруктове пюре  |           | 2             | 40                  |

$x1 + x2 →max $
